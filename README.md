# Typing Speed Test

A responsive Typing Speed Test web application built using HTML, CSS, and JavaScript that measures typing speed (WPM), accuracy, and performance through a real-time countdown timer and an interactive user interface.

## Features

- Measure typing speed (Words Per Minute)
- Calculate typing accuracy
- Real-time countdown timer
- Instant typing performance results
- Restart test functionality
- Responsive and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://madhuri-yelure.github.io/Typing-Speed-Test/


## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
